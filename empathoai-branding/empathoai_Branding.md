\## TYPOGRAPHY SYSTEM — LOGO (LOCKED)



\### Brand Name



EmpathoAI  

\- E uppercase  

\- mpatho lowercase  

\- AI uppercase  

\- No variations allowed  



\### Font



\- Helvetica Now (only)



\### Structure



\*\*Empatho\*\*

\- Weight: Bold

\- Color: #F5F5F5

\- Role: Primary (thinking, diagnosis)



\*\*AI\*\*

\- Weight: Regular

\- Color: Gray (#7A8087 – #8A8F98)

\- Role: Secondary (system layer)



Rules:

\- AI must NEVER dominate Empatho

\- Always read as secondary layer



\### Tagline



Text: Growth Architecture  

\- Weight: Light  

\- Letter spacing: 800  

\- Color: #C56A1A  



Role:

\- Conceptual descriptor (execution system)



\### Hierarchy



1\. Empatho  

2\. AI  

3\. Growth Architecture  



If broken → invalid



\### Composition & Alignment

- Stacking: Vertically stacked (Logo top, Tagline bottom).
- Internal Alignment: Tagline MUST be centered relative to the visual width of "EmpathoAI".
- Global Positioning: The entire composition unit must be left-aligned in the layout (use w-fit container logic).
- Width Balance: Letter spacing of "Growth Architecture" should be adjusted so its start and end points align visually with the logo width (approx. 0.41em tracking).

\### Forbidden

- Change font
- Change casing
- Swap weights
- Apply effects
- Use orange as background
