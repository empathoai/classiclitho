# Logo System for EmpathoAI

> Brand asset rules for premium presentations, proposals, documents, and web usage.

## 1. Purpose

EmpathoAI uses two primary logo lockups:
- **Primary lockup with tagline**
- **Primary lockup without tagline**

This is intentional.
The tagline version positions the brand.
The non-tagline version preserves visual purity in repeated or functional contexts.

## 2. Official Files

### With Tagline
- `logo-primary-tagline-dark.svg`
- `logo-primary-tagline-light.svg`

### Without Tagline
- `logo-primary-dark.svg`
- `logo-primary-light.svg`

## 3. Naming Logic

- **primary** = official master brand lockup
- **tagline** = includes `Growth Architecture`
- **dark** = use on dark backgrounds
- **light** = use on light backgrounds

## 4. Role of Each Version

### 4.1 Primary Lockup with Tagline
Use when the brand needs positioning or introduction.

Best for:
- presentation covers
- first page of proposals
- institutional documents
- brand introduction moments
- website hero or about sections when category clarity matters

Function:
- introduces the brand
- reinforces strategic positioning
- explains category through `Growth Architecture`

### 4.2 Primary Lockup without Tagline
Use when the brand is already introduced or when the layout needs more restraint.

Best for:
- internal slides after the cover
- final slide signatures
- headers
- footers
- repeated brand presence
- tighter or more premium compositions

Function:
- preserves visual cleanliness
- reduces noise
- keeps the system feeling confident and controlled

## 5. Usage Rule

### Recommended Premium Rule
- **First appearance of the brand** -> use logo **with tagline**
- **Subsequent appearances** -> use logo **without tagline**

This is the default rule for EmpathoAI presentations.

## 6. Background Rules

### Dark Versions
Use on:
- black backgrounds
- dark charcoal surfaces
- dark presentation covers
- dark website sections

Official file:
- `logo-primary-dark.svg`
- `logo-primary-tagline-dark.svg`

### Light Versions
Use on:
- white backgrounds
- light gray documents
- white proposals
- light sections with high contrast

Official file:
- `logo-primary-light.svg`
- `logo-primary-tagline-light.svg`

## 7. Composition Rules from Brand Identity

The logo system must preserve these non-negotiable rules:

### Wordmark Construction
- `E` uppercase
- `mpatho` lowercase
- `AI` uppercase

### Typography
- Helvetica Now only

### Weight Logic
- `Empatho` -> Bold
- `AI` -> Regular
- `Growth Architecture` -> Light

### Color Logic
- `Empatho` -> clarity / primary emphasis
- `AI` -> secondary system layer
- `Growth Architecture` -> decision accent / positioning layer

### Alignment Logic
- Logo and tagline are vertically structured in the tagline lockup
- Tagline should remain centered relative to logo width
- Entire lockup should sit left-aligned in layouts by default

## 8. Presentation-Specific Rules

### Cover Slide
Preferred:
- `logo-primary-tagline-dark.svg` on dark premium covers
- `logo-primary-tagline-light.svg` on light premium covers

### Middle Slides
Preferred:
- no logo at all, or
- a small restrained version of `logo-primary-dark.svg` / `logo-primary-light.svg`

### Final Slide
Preferred:
- `logo-primary-dark.svg` or `logo-primary-light.svg`
- use tagline again only if the slide is intended as a positioning close, not just a contact close

## 9. Size Guidance

### With Tagline
Use when there is enough room for the tagline to remain crisp and intentional.
Do not shrink the tagline version until the tagline becomes hard to read.
If readability is compromised, switch to the non-tagline version.

### Without Tagline
Preferred for smaller placements and repeated use.
This version scales better in compact layouts.

## 10. Clear Space Rule

Maintain a minimum clear space around the logo equal to approximately the height of the `AI` block or a generous equivalent visual margin.

Rule of thumb:
- never let text, lines, charts, or page edges crowd the logo
- in premium layouts, give it more space than you think it needs

## 11. Forbidden Uses

Do not:
- change the casing
- change the typeface
- swap weights
- recolor freely outside approved variants
- add shadows, glows, gradients, or effects
- stretch or distort the logo
- place the logo over visually noisy backgrounds
- use the tagline version repeatedly across many slides
- use orange as a full background behind the logo

## 13. Source Identity Rule

The logo should feel like:
- strategic
- controlled
- premium
- certain

If the logo treatment feels loud, decorative, startup-like, or overexposed, it is being used incorrectly.
